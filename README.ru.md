# TSCRT

**Работает на Windows, Linux и macOS.**

**TSCRT** — кроссплатформенный эмулятор терминала SSH и последовательной консоли, построенный на Qt 6, libssh2 и libvterm. Он предназначен для инженеров, которым нужна надёжная, скриптуемая многосессионная консоль для работы со встраиваемыми устройствами, сетевым оборудованием и удалёнными серверами.

## Возможности

### Сессии и терминал
- Сессии **SSH** на базе libssh2 с keepalive, автоматическим переподключением и **проверкой ключа хоста (TOFU)**
- Сессии **Serial / RS-232** для встраиваемых целей
- Управление многовкладочными сессиями с сохраняемыми профилями
- **Тип терминала для каждой сессии** (xterm-256color, vt100 и т. п.) с глобальным значением по умолчанию
- **Разделённые панели** (горизонтально/вертикально) для каждой вкладки
- **Трансляция ввода** на все панели вкладки (`Ctrl+Shift+B`)
- **Панель поиска** с regex и учётом регистра по всему скроллбэку (`Ctrl+F`)
- Постоянная подсветка на уровне всего терминала / функция **Mark**
- Полноэкранный режим (`F11`) — все горячие клавиши остаются активны

### Многооконность
- **Отделение вкладки в новое окно** — щёлкните правой кнопкой на вкладке и выберите «Отделить в новое окно», чтобы переместить её в отдельный экземпляр TSCRT, сохраняя соединение
- **Перетаскивание вкладок между окнами** — перетащите вкладку с панели одного окна на другое
- Контекстное меню вкладки: Переименовать, Дублировать, Закрепить, Запустить снимок, Отделить, Закрыть

### Автоматизация
- **Движок автоматизации** — действия при запуске, действия по шаблону (с ограничением частоты), периодические задачи
- **Снимки (Snapshots)** — выполняйте shell-команды при подключении, по шаблону или по расписанию и фиксируйте вывод
- **Планировщик Cron** — 5-польные cron-выражения для времени снимков
- **Панель кнопок** — настраиваемые для каждой сессии кнопки быстрых действий
- **Виджет командной строки** для inline-выполнения действий / скриптов

### Безопасность
- **Проверка ключа хоста SSH** — модель TOFU с файлом `known_hosts`; несовпадающие ключи отклоняются жёстко
- Учётные данные зашифрованы в хранении: **DPAPI** (Windows), **Keychain** (macOS), fallback в открытом виде (Linux)
- Профиль Windows защищён **ACL только для владельца**
- SMTP: **валидация TLS-сертификата обязательна**, AUTH в открытом виде отклоняется, инъекция заголовков блокируется
- Дампы сбоев: права только для владельца, автоудаление через 7 дней
- Триггеры автоматизации: ограничение частоты для каждого шаблона и общее
- Переподключение: минимальная базовая задержка 250 мс, жёсткий лимит 100 попыток

### Логирование и надёжность
- Логгер сессий для каждой панели с отдельным диалогом настроек логов
- **SMTP-клиент** для уведомлений по электронной почте (STARTTLS/SMTPS требуется для аутентификации)
- **Обработчик сбоев** с поддержкой отладочных символов
- Автопереподключение с экспоненциальной задержкой и джиттером

### Интернационализация
- Встроенные переводы интерфейса: **English**, **한국어**, **日本語**, **中文**, **Deutsch**, **Español**, **Français**, **Italiano**, **Português**, **Русский**
- Встроенная HTML-справка для каждого языка

## Требования

| Компонент | Версия |
|-----------|---------|
| CMake     | >= 3.20  |
| C / C++   | C11 / C++17 |
| Qt        | >= 6.5 (Core, Gui, Widgets, Network, LinguistTools, Test) |
| libssh2   | последняя  |
| libvterm  | последняя  |
| pkg-config | обязателен |

**Windows**: тулчейн MSYS2 UCRT64 (gcc 15+).
**macOS**: 12+ с Homebrew.

## Сборка — Windows (MSYS2 UCRT64)

```bash
pacman -S --needed \
  mingw-w64-ucrt-x86_64-{gcc,cmake,ninja,pkgconf,qt6-base,qt6-tools,libssh2,libvterm}

source env.sh
./build.sh              # инкрементальная (для полной пересборки используйте `clean`)
./install.sh            # развёртывание exe + Qt DLL
./install.sh package    # NSIS-установщик
```

## Сборка — macOS

См. [`mac_build.txt`](mac_build.txt).

```bash
brew install qt@6 libssh2 libvterm pkg-config cmake ninja
cmake -S . -B build -G Ninja -DCMAKE_BUILD_TYPE=Release
cmake --build build
cd build && cpack -G DragNDrop
```
