TSCRT 使用指南

TSCRT 是一款面向 SSH2 与串口控制台的终端仿真器。主要功能导航：

  • 会话
  • 标签页 & 多窗口
  • 分割窗格 / 输入广播 / 自动重连
  • 按钮（含示例）
  • 快照（含示例）
  • 自动化规则
  • 触发器 / 定时执行 / 启动动作
  • SMTP（邮件发送）
  • 日志
  • 安全
  • 配置文件导入 / 导出
  • 快捷键

会话

会话是一个 SSH 或串口连接定义。

  • Sessions → Manage sessions… — 添加、编辑、复制、删除会话。左侧停靠栏以树形展示。
  • SSH：主机、端口、用户名、密码或私钥文件。
  • 串口：设备名（如 COM3）、波特率、数据/停止/校验位、流控。

会话编辑器（三标签页）

  • Connection — 连接参数。
  • General — 会话名、连接类型、每会话终端类型（留空 = 使用全局默认值）、日志设置、全屏显示选项。
  • Advanced — 自动重连设置、SSH keepalive。

标签页 & 多窗口

  • Ctrl+Alt+← / → — 上/下一个标签页。
  • F11 — 全屏切换。所有快捷键在全屏下均可用。

标签页右键菜单

  • Rename / Duplicate / Run Snapshot / Pin
  • Detach to New Window — 将标签页分离到新 TSCRT 窗口（保持连接）。

窗口间拖拽标签页

拖拽标签页到另一个 TSCRT 窗口的标签栏即可转移会话；拖到窗口外部则自动创建新窗口。连接不会中断。

分割窗格 / 输入广播 / 自动重连

分割窗格

  • Ctrl+Shift+H — 水平分割。Ctrl+Shift+V — 垂直分割。
  • Ctrl+Shift+W — 关闭活动窗格。
  • 活动窗格以蓝色边框标识。

输入广播

  • Ctrl+Shift+B 切换。全部窗格显示橙色边框。按标签页生效。

自动重连

  • Max attempts：硬上限 100 次。Base delay：最小 250 ms，指数退避，上限 30 s。
  • 滚动历史保留。每个窗格独立重连状态。

按钮

最多 20 个用户自定义按钮。动作字符串：\r=回车, \n=换行, \t=制表符, \b=退格, \e=ESC, \p=暂停1秒, \\=\。未定义的转义序列被忽略。

Loop：右键按钮 → 定时重复。Mark：在日志中插入时间戳标记。

快照

Snapshots → Manage snapshots… 配置。expect_prompt 在 64 KB 滚动窗口中匹配。

执行：菜单或标签页右键 → Run Snapshot。

自动化规则

on_connect / cron（5字段） / pattern（正则+冷却时间）。

触发器 / 定时执行 / 启动动作

Settings → Preferences。触发器有速率限制：每模式 2 秒冷却，每会话 10 秒最多 20 次。

SMTP

  • 配置用户名时必须启用 STARTTLS/SMTPS（拒绝明文认证）。
  • 密码加密存储（Windows: DPAPI，macOS: Keychain）。
  • 强制 TLS 证书验证。

日志

Logs → Open log folder。快照：snapshots/。日志开关在 General 标签页。

安全

SSH 主机密钥验证（TOFU）

首次连接时固定到 ~/tscrt/known_hosts。不匹配时拒绝连接（防御中间人攻击）。

凭据存储

Windows: DPAPI + owner-only ACL。macOS: 钥匙串。Linux: 明文（0600）。

崩溃转储

owner-only 权限。超过 7 天自动删除。

配置文件

File → Export/Import profile…
