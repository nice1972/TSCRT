Руководство пользователя TSCRT

TSCRT — эмулятор терминала для SSH2 и последовательных консолей.

Сессии

Редактор с тремя вкладками: Connection, General (тип терминала для каждой сессии), Advanced.

Вкладки и многооконность

Ctrl+Alt+←/→ навигация. F11 полноэкранный режим. Правый клик: Переименовать, Дублировать, Закрепить, Отделить в новое окно. Перетаскивание вкладок между окнами.

Разделённые панели / Трансляция / Переподключение

Ctrl+Shift+H/V/W разделить/закрыть. B трансляция ввода (оранжевая рамка). Переподключение: макс. 100 попыток, мин. задержка 250 мс, потолок 30 с.

Кнопки

Макс. 20. \r=Enter, \e=ESC, \p=пауза 1с. Неизвестные escape-последовательности игнорируются.

Снимки

Snapshots → Manage… expect_prompt в скользящем окне 64 КБ.

Правила автоматизации

on_connect / cron / pattern (+ период охлаждения).

Триггеры / Периодические / Запуск

Ограничение: 2 с на шаблон, макс. 20 за 10 с на сессию.

SMTP

STARTTLS/SMTPS обязательно. Проверка TLS принудительная. Пароли зашифрованы (DPAPI/Keychain).

Журналы

Logs → Open log folder.

Безопасность

Проверка SSH-ключа хоста (TOFU). Учётные данные: DPAPI/Keychain. Дампы: права только для владельца, авточистка через 7 дней.
